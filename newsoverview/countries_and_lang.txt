Information:
For GoogleNews to work, it must be 'activated' by selecting a country and a language. I have put
the default as country = 'GB', lang = 'en'.
Once the country and language (lang) have been selected, you can enter the search query in the
selected language, and it will return the results back in that language.

On this page, the 1st part consists of the commonly used countries with their respective
language(s).
The 2nd part contains a list of countries which support English as the input language. Other
languages spoken by that country that will work with GoogleNews are also noted.
The 3rd part contains a list of all countries.
The 4th part contains a list of all languages.
